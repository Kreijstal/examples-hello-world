Deno is a runtime for JavaScript and TypeScript that is based on the V8 JavaScript engine and the Rust programming language. It was created by Ryan Dahl, the original creator of Node.js.

Features
Secure by default: no file, network, or environment access unless explicitly enabled
Supports TypeScript out of the box
Ships as a single executable
Built-in tooling: formatter, linter, test runner, bundler
Deno Deploy

Deno Deploy is a serverless platform for running JavaScript and TypeScript at the edge. It is built on the Deno runtime and uses V8 isolates for fast cold starts.
