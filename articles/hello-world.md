A "Hello, World!" program is generally a simple computer program which outputs (or displays) to the screen a message similar to "Hello, World!".

## History

The first known "Hello, World!" program was written by Brian Kernighan in 1972 as part of the documentation for the B programming language.

## Examples

In Python:

```python
print("Hello, World!")
```

In JavaScript:

```javascript
console.log("Hello, World!");
```

In Deno (TypeScript):

```typescript
console.log("Hello, World!");
```
